# Projeto 4 - Prevendo Customer Churn em Operadoras de Telecom

Customer Churn (ou Rotatividade de Clientes, em uma tradução livre) refere-se a uma decisão tomada pelo cliente sobre o término do relacionamento comercial. Refere-se também à perda de clientes. A fidelidade do cliente e a rotatividade de clientes sempre somam 100%. Se uma empresa tem uma taxa de fidelidade de 60%, então a taxa de perda de clientes é de 40%. De acordo com a regra de lucratividade do cliente 80/20, 20% dos clientes estão gerando 80% da receita. Portanto, é muito importante prever os usuários que provavelmente abandonarão o relacionamento comercial e os fatores que afetam as decisões docliente. 
Este trabalho é parte do curso **Big Data Real-Time Analytics com Python e Spark da formação Cientista de Dados da Data Science Academy** cujo objetivo é prever o Customer Churn em uma Operadora de Telecom, ou seja um modelo para prever a probabilidade do cliente cancelar o plano
